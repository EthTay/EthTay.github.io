What is EDGE?

EDGE is a project to numerically simulate the very smallest galaxies in the Universe. It is an international collaboration spear headeded by the University of Surrey (PI Prof. Justin Read), Durham University and the University of Lund. The simulations are set up using the new GenetIC initial conditions generator and run using the RAMSES code. Analysis of simulation outputs is performed using the AHF halo finder in addition to both PyNbody and Tangoes analysis tools. The EDGE simulations are run on the DiRAC High Performance Computing Facility hosted across the UK and has been awarded 128.35 MCPU hours apart of project codes dp101, dp191, dp324

While EDGE is an international effort, we would not be here without the hard work of all who put in their time to work on these simulations. For a full list of amazing people apart of EDGE please see the Collaborators page.
